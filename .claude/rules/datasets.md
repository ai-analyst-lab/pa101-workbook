# Working with a dataset

The `definitions.md` beside the data is the record. Do not substitute your own definition of a
metric, and do not quote a rate without naming the definition that produced it.

Check the cautions section of that file before using any column. Some columns are measured after the
behaviour being studied and cannot be used as evidence about what preceded it.

If you need a definition that is not in the file, say so rather than inventing one.
